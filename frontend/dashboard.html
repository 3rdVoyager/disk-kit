Getting Started

Set up your workspace and run your first tool in a few minutes.

  1. 1
    Choose a default folder

    Open Settings and pick where Disk Kit should work by default. You can also paste a path directly into any tool.

  2. 2
    Run a tool

    Convert images, batch-rename files, or find exact duplicate copies — everything runs locally on your machine.

  3. 3
    Review your history

    Completed runs appear below in Recent Operations so you can track what Disk Kit has done.
